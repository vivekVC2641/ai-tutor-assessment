# Mini AI Tutor - Agent Enhancements

This file lists proposed agent enhancements for the current system in a simple format.

## Agent Enhancements

### 1) Master Orchestrator Agent
Routes work dynamically between tutor, evaluator, and support agents based on context and confidence.
This improves decision quality compared to fixed linear flow.

### 2) Socratic Tutor Agent
Provides guided hints and questions before giving a full answer.
This improves conceptual understanding and student retention.

### 3) Assessment Agent
Adds rubric-based scoring with clearer structured feedback.
This keeps grading more consistent and explainable.

### 4) Diagnostic Agent
Detects weak concepts and repeated mistake patterns from student responses.
This helps create targeted remediation instead of generic feedback.

### 5) Planning Agent
Builds adaptive weekly study plans from gap analysis and progress signals.
This makes learning paths personalized and measurable.

### 6) Content Agent
Selects best-fit learning material and generates additional practice questions where needed.
This improves content coverage and practice quality.
